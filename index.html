Шпаргалка по Git


Команда
Описание

git init
Cоздает новый репозиторий Git

git clone url
Утилита командной строки Git для выбора существующего репозитория и создания его клона

git add file
Первая команда, предписывающая Git «сохранить» снимок текущего состояния проекта в истории коммитов

git add .
Перенос ожидающих изменений из рабочего каталога в раздел проиндексированных файлов Git

git commit -m "comment"
Позволяет написать сообщение в строке с командой «commit»

git commit -a -m "comment"
Позволяет добавить и создать сообщение для коммита одной команды

git push url
Позволяет отправлять локальную ветку на удаленный репозиторий

git status
Отображает состояние рабочего каталога и раздела проиндексированных файлов

git log
Перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке

git branch branch_name
Позволяет создавать, просматривать, переименовывать и удалять ветки

git checkout branch_name
Позволяет перемещаться между ветками, созданными командой git branch

git checkout -b branch_name
Cоздает ветку с указанным именем и автоматически переключится на неё

git remote
Позволяет создавать, просматривать и удалять подключения к другим репозиториям
